# 🖼️ Desktop Wallpaper Viewer

A lightweight GUI application built using Python and Tkinter that allows users to cycle through local wallpapers seamlessly.

---

## ✨ Features

* **Image Resizing:** Automatically resizes images using Pillow (PIL) to fit the window cleanly.
* **Wallpaper Rotation:** Cycle through all images inside the `Wallpapers/` directory using a simple control button.
* **Dynamic Path Handling:** Automatically resolves pathing relative to the script location.

---

## 🚀 Getting Started

### Prerequisites

* Python 3.x
* `Pillow` (PIL) library
